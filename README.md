# acert-web
website 

## How to use

clone the repository

set up with npm :

```bash
npm install
npm run develop 
```

or yarn:

```bash
yarn
yarn dev
```
## add or modify content with prisma

open prisma studio :

```bash
npx prisma studio
```
or:

```bash
yarn prisma studio
```

note:
- images use the path from public (example if cat.jpg is in the pulic dir, enter /cat.jpg)
